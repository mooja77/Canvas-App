# Move Qualitative Projects with QDPX | QualCanvas Import and Export

**Outcome:** Use QDPX to transfer a qualitative project, then verify the result in the destination tool.

**Audience:** Not made for kids  
**Language:** English (Ireland)  
**Voice:** en-IE-EmilyNeural

## Scene plan

### 1. Move a project without treating export as magic (0:00)

- Visual: Original QualCanvas brand frame
- On-screen callout: QDPX is an exchange format—not a guarantee of perfect parity
- Narration: QDPX is an open exchange format for qualitative projects. QualCanvas can export and import QDPX, which helps when work needs to move to or from a compatible tool. Always plan time to verify the round trip.

### 2. Open Export and import (0:17)

- Visual: captures/20-export-menu.png
- On-screen callout: Choose QDPX when another qualitative tool needs the project
- Narration: Open the Export and import menu. Use QDPX for project exchange, while reports, spreadsheets and images serve different handoff needs. Save the export in an approved research location.

### 3. Import into an existing canvas (0:33)

- Visual: captures/34-qdpx-import.png
- On-screen callout: Select a QDPX or ZIP package of no more than twenty megabytes
- Narration: For import, choose Import QDPX and select a QDPX or ZIP package within the displayed twenty-megabyte limit. Read the warning in the dialog before changing an existing canvas.

### 4. Use a copy for the first transfer (0:47)

- Visual: captures/34-qdpx-import.png
- On-screen callout: Protect the source project before a merge or replacement
- Narration: For the first transfer, work in a copy or fresh destination canvas. That gives you a recoverable comparison if a source type, code relation or formatting detail maps differently between applications.

### 5. Verify what arrived (1:01)

- Visual: captures/23-codebook.png
- On-screen callout: Sources, excerpts, codes, definitions, hierarchy and memos
- Narration: After import, compare source counts, open sample excerpts and inspect the codebook, hierarchy and memos. Features that are unique to either application may not have a one-to-one equivalent in the exchange format.

### 6. Treat the package as research data (1:17)

- Visual: captures/09-privacy.png
- On-screen callout: Transfer, storage and recipient controls still apply
- Narration: A QDPX package can contain participant text and analytical notes. Encrypt or otherwise protect it where your protocol requires, send it only to authorised recipients and delete temporary copies according to your retention plan.

### 7. Exchange, then verify (1:33)

- Visual: Original QualCanvas brand frame
- On-screen callout: Back up · transfer · inspect · reconcile · document
- Narration: A responsible QDPX handoff includes a backup, a controlled transfer, structured verification and a note of anything that changed. The QualCanvas training centre includes separate tutorials for reports, spreadsheets and visual exports.
