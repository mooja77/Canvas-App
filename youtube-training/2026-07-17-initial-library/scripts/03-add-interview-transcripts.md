# Add Interview Transcripts in QualCanvas | Import, Organise & Protect Data

**Outcome:** Add a transcript, keep sources organised and apply the right privacy checks before import.

**Audience:** Not made for kids  
**Language:** English (Ireland)  
**Voice:** en-IE-EmilyNeural

## Scene plan

### 1. Add a transcript without losing context (0:00)

- Visual: Original QualCanvas brand frame
- On-screen callout: Use synthetic data for practice; use approved data for real studies
- Narration: A transcript is more than a block of text. Before import, confirm that the study permits this storage location, remove direct identifiers where appropriate and use synthetic material whenever you are only learning the interface.

### 2. Check the supported workflow (0:15)

- Visual: captures/04-guide.png
- On-screen callout: The guide covers adding, reading and coding sources
- Narration: The public guide explains the supported source workflow and current limits. Review it before a large import so file types, transcript length and plan limits do not interrupt the study.

### 3. Start from the Transcript action (0:28)

- Visual: captures/14-transcript-menu.png
- On-screen callout: Paste text or choose an available import path
- Narration: In the workspace, open Transcript. Choose the path that matches your material, give the source a useful title and keep participant identifiers out of the title unless the protocol explicitly allows them.

### 4. Use Sources to stay oriented (0:42)

- Visual: captures/30-sources-panel.png
- On-screen callout: See what is present before opening or adding more
- Narration: The Sources tab provides a compact inventory. Use it to confirm that each interview arrived once, spot naming inconsistencies and return to a particular source without searching the whole canvas.

### 5. Read in context (0:56)

- Visual: captures/13-canvas-workspace.png
- On-screen callout: The full passage remains visible beside its codes
- Narration: Open the transcript node and read around each passage before coding. Qualitative meaning depends on context, so avoid creating a code from a highlighted phrase alone when the surrounding answer changes its interpretation.

### 6. Treat research data as research data (1:11)

- Visual: captures/09-privacy.png
- On-screen callout: Review retention, deletion, portability and processor locations
- Narration: For real participant material, align QualCanvas with the consent form, ethics approval and institutional policy. The privacy page documents retention, deletion, portability and service-provider locations; your protocol remains the controlling boundary.

### 7. Import deliberately (1:30)

- Visual: Original QualCanvas brand frame
- On-screen callout: Check permission · remove identifiers · name clearly · verify once
- Narration: The safe habit is straightforward: check permission, remove unnecessary identifiers, name the source clearly and verify the import once. Then you are ready to code.
