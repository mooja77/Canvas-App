# Export QualCanvas Work for a Dissertation, Report or Handoff

**Outcome:** Choose an export format that preserves the evidence and context needed by the next reader.

**Audience:** Not made for kids  
**Language:** English (Ireland)  
**Voice:** en-IE-EmilyNeural

## Scene plan

### 1. Export for the next reader (0:00)

- Visual: Original QualCanvas brand frame
- On-screen callout: Choose the format by what they need to verify or continue
- Narration: An export is a handoff, not just a download. Decide whether the next person needs to read the argument, inspect the codebook, reuse the project in another tool or see the visual structure.

### 2. Use the Export menu (0:13)

- Visual: captures/20-export-menu.png
- On-screen callout: Image · report · Excel · disclosure · QDPX
- Narration: Open Export and choose the format by purpose. A canvas image is useful for slides. A report supports reading. Excel supports tabular review. QDPX supports exchange with compatible qualitative-analysis software.

### 3. Export a codebook people can interpret (0:33)

- Visual: captures/23-codebook.png
- On-screen callout: Include definitions and examples—not names alone
- Narration: Before handing off a codebook, review names, definitions, hierarchy and examples. A list of labels without decision rules gives the next coder very little chance of applying them consistently.

### 4. Fit the canvas before exporting an image (0:47)

- Visual: captures/13-canvas-workspace.png
- On-screen callout: Make nodes readable at normal viewing size
- Narration: For a visual export, fit the relevant nodes into view and remove unnecessary empty space. Check the image at ordinary slide or document size; tiny transcript text is not useful evidence.

### 5. Check format details in the guide (1:02)

- Visual: captures/04-guide.png
- On-screen callout: Supported outputs and limits can change
- Narration: The guide documents the current export choices and their intended use. Review it before a deadline, especially when another application, committee or archive requires a specific format.

### 6. Protect the exported copy (1:16)

- Visual: captures/09-privacy.png
- On-screen callout: An export inherits the sensitivity of the source data
- Narration: Downloaded files are outside QualCanvas access controls. Store them in the approved location, limit onward sharing and apply the study's retention and deletion rules to every copy.

### 7. A useful export preserves context (1:29)

- Visual: Original QualCanvas brand frame
- On-screen callout: Purpose · format · readability · secure destination
- Narration: Before exporting, confirm the purpose, choose the right format, test readability and send the file to an approved destination. That turns a download into a responsible handoff.
