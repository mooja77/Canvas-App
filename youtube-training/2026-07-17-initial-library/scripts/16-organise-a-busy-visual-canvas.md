# Organise a Busy QualCanvas Workspace | Layout and Shortcuts

**Outcome:** Use auto-arrange, fit view, navigation, display controls and the command palette to manage a growing canvas.

**Audience:** Not made for kids  
**Language:** English (Ireland)  
**Voice:** en-IE-EmilyNeural

## Scene plan

### 1. Keep a growing canvas readable (0:00)

- Visual: Original QualCanvas brand frame
- On-screen callout: Organisation supports thinking when the project becomes busy
- Narration: A visual canvas becomes more valuable as the analysis grows, but only if you can still find the relevant evidence. QualCanvas includes layout and navigation controls that help without changing your coding decisions.

### 2. Use Fit View to recover the whole picture (0:15)

- Visual: captures/13-canvas-workspace.png
- On-screen callout: Zoom out, orient yourself, then move to the relevant cluster
- Narration: Choose Fit View when nodes have moved beyond the visible area. Start from the full structure, then zoom or pan toward the source, code or memo you need instead of repeatedly dragging the canvas at random.

### 3. Use Auto-arrange as a layout tool (0:29)

- Visual: captures/35-more-actions.png
- On-screen callout: More canvas actions → Auto-arrange layout
- Narration: Open More canvas actions and choose Auto-arrange layout when the workspace needs a clean starting point. Auto-arrange changes positions; it does not decide what your evidence means or rewrite the analytical links you created.

### 4. Open commands with Control K (0:44)

- Visual: captures/22-command-palette.png
- On-screen callout: Search common actions instead of hunting through menus
- Narration: Press Control K to open the command palette. Search for a canvas action, run it and return to the analysis. Keyboard shortcuts are especially useful during repeated coding and review sessions.

### 5. Navigate from the source list (0:59)

- Visual: captures/30-sources-panel.png
- On-screen callout: Use Sources to return to the material behind the map
- Narration: Open Sources when you need a stable route back to an interview or response. The visual layout should support the evidence trail, not replace careful reading of the source material.

### 6. Choose a comfortable display (1:12)

- Visual: captures/28-dark-mode.png
- On-screen callout: Light and dark modes affect display—not the project data
- Narration: Switch between light and dark display modes for the working environment that suits you. On a phone, use the canvas for quick review; detailed visual organisation is clearer on a larger screen.

### 7. A tidy view should preserve the trail (1:25)

- Visual: Original QualCanvas brand frame
- On-screen callout: Fit · arrange · search · navigate · review
- Narration: Fit the view, arrange when useful, search for commands and keep returning to the source. Continue in the training centre with the repository tutorial for organising insights across more than one project.
