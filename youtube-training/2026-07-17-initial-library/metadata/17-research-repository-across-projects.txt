TITLE
Build a Research Repository Across QualCanvas Projects

DESCRIPTION
Create a research repository and organise carefully written insights across projects while preserving links back to the source analysis.

QualCanvas keeps transcripts, codes, memos and analysis views connected in a visual workspace, so the reasoning behind a qualitative interpretation remains reviewable.

Start QualCanvas: https://qualcanvas.com
