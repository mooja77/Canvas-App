TITLE
Import Open-Text Survey Data into QualCanvas

DESCRIPTION
Import a CSV of open-text responses, map useful columns and prepare responses for coding without exposing respondent data.

QualCanvas keeps transcripts, codes, memos and analysis views connected in a visual workspace, so the reasoning behind a qualitative interpretation remains reviewable.
