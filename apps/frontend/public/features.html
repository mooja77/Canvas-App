Features

Everything QualCanvas does for qualitative analysis

A visual coding tool built for researchers who want NVivo's analytical depth without the learning curve. Spatial canvases, link evidence to themes, share with collaborators in real time.

Try free for 14 days

Coding

  • Visual canvasDrag interview quotes, observations, and documents onto a canvas. Group them spatially. Re-arrange as your understanding evolves.
  • Theme nestingThemes contain sub-themes contain quotes. Unlimited depth, no rigid hierarchy.
  • Inline codingHighlight a passage in a transcript, code it. The code appears as a labelled card on your canvas, linked back to the source.
  • Multi-source evidenceOne theme can link to interview quotes, document excerpts, photos, and field notes. Spatial layout shows the depth of your support.

Collaboration

  • Real-time canvasesTwo researchers code the same canvas simultaneously. See each other's cursor and edits live.
  • Code agreement reportsHow often did two coders apply the same code to the same passage? QualCanvas shows you inter-rater reliability with no extra setup.
  • Permission rolesLead researcher / contributor / read-only. Useful for thesis supervisors and student researchers.
  • Comments and decisionsDiscuss themes inside the canvas. Decisions are logged so you can audit the analytical trail.

Analysis output

  • Theme map exportPublication-ready theme maps as PNG/SVG. Drop them straight into your paper.
  • Quote-by-theme tablesAuto-generated tables of every quote per theme, ready for the appendix.
  • Frequency reportsHow often each theme appeared, by participant or document. Useful for mixed-methods triangulation.
  • Decision audit logEvery coding decision and its timestamp. Required by IRB / ethics boards in many institutions.
