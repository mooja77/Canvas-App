# QualCanvas — Outreach Email Templates

Four cold-email templates, each <120 words, each tuned to a single buyer persona. All templates leave the personalization slot `{{...}}` deliberately small — if you can't fill in a _specific_ opening line about the recipient (their paper, their dept, their LinkedIn post), don't send the email. Generic mass mailers are why qualitative researchers have spam filters tuned to "research software."

Subject lines: kept under 50 chars so they don't truncate on mobile. A/B test the top two if volume justifies it.

## Personalization checklist (before hitting send)

- [ ] First name is correct and spelled right
- [ ] The "specific opening" line names a real thing (paper title, dept name, recent post) — not a generic platitude
- [ ] University / firm name in body matches what they actually use (e.g., "King's" not "King's College London")
- [ ] If template references a paper, the title is exact (academics will catch a paraphrase and discard you)
- [ ] No follow-up automation for 14 days; if no reply, one single follow-up only, then stop

## Response targets

- **Open rate target:** 35%+ (industry: ~22%)
- **Reply rate target:** 8%+ (industry: ~1%)
- **Trial-signup rate target:** 3%+ of opens
- **Trial-to-paid conversion target:** 25%+ on .edu emails (with academic discount applied)

If any of these fall below half target, the issue is almost always the personalization slot, not the body. Re-read the personalization checklist before iterating on the body copy.
