# 6.18 `/press` — press / media kit page

← [Pages index](README.md) · [Plan index](../README.md)

`/press` — single page with downloadable assets and a brief fact sheet. Used by journalists, library guides, conference programs.

## Content

- Eyebrow: `PRESS`. H1 Fraunces: `Press & media.`
- One-paragraph positioning sentence (matching [01 §1.1](../01-strategic-positioning.md)).
- **About** — 2 short paragraphs on what QualCanvas is, who built it (one mention of JMS Dev Lab in Ireland), when it launched.
- **Fact sheet** — Founded year, headquarters, founder count, customer count (when meaningful), free tier availability, integrations (NVivo QDPX, ATLAS.ti QDPX), languages (en / es / fr / de in app).
- **Brand assets** — single downloadable `.zip`: logo SVG light + dark, wordmark, color values as text, screenshot set at 320 / 640 / 1280 / 1920 px, brand do/don't quick rules card.
- **Sample screenshots** — full-resolution AVIFs for press use, with captions describing what each shows.
- **Quotes for citation** — one paragraph each from the founder and one researcher (with explicit permission documented — see [10](../10-risks-and-rollback.md) R20).
- **Contact** — `press@qualcanvas.com`.

## Layout

Single page, no chrome, no FAQ accordion. Asset cards in a 3-up grid with download buttons.

## Read time

~3 min.
