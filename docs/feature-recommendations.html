Strategic Position

Our Competitive Moat

🎨 Web-Native Node-Graph Canvas

No other web-based QDA tool has a React Flow node-graph interface. ATLAS.ti's network views and MAXQDA's MAXMaps are desktop-only. Quirkos has visual bubbles but no relationships. This is our biggest differentiator.

💰 10x Cheaper Than Incumbents

At $12/mo (Pro) and $29/mo (Team), we're 90% cheaper than NVivo ($1,069/yr) and ATLAS.ti ($588/yr). Only Dedoose ($19/mo) and Delve ($19/mo) are comparably priced — and they lack our visual canvas.

🔗 Semantic Code Relationships

Our relation edges (causes, supports, contradicts, etc.) on the canvas are a native advantage. Only ATLAS.ti offers comparable semantic code-to-code linking. Doubling down here creates a deep moat.

⚡ Modern Stack, Fast Iteration

React + Vite + Prisma + Tailwind means we can ship features 5-10x faster than legacy desktop apps rewriting for the cloud. NVivo and MAXQDA are decades-old C++/Java codebases.

What to Build Next

🔴 Critical — Close the Gap
🤖

AI-Assisted Coding

When a user highlights text, suggest relevant existing codes and new code names via LLM. Auto-code entire uncoded transcripts with human review. Show AI-suggested codes as dashed-border nodes on canvas.
Who has it: ATLAS.ti (GPT-4), NVivo, MAXQDA, Dovetail, Delve
LLM Table Stakes by 2025
User Impact95%
🎤

Audio/Video Transcription

Upload audio/video files and auto-transcribe using Whisper API. Create transcript nodes automatically with timestamps. Clicking a coded excerpt plays the corresponding audio segment.
Who has it: Dovetail, ATLAS.ti, NVivo, MAXQDA (all AI-powered)
Whisper API Workflow Blocker
User Impact90%
👥

Real-Time Collaborative Canvas

WebSocket-based multi-user canvas editing. See teammates' cursors, live code applications, and node movements. Presence indicators in the toolbar. Conflict-free CRDT-based state sync.
Who has it: Dedoose, Dovetail, ATLAS.ti Cloud
WebSocket Team Tier Must-Have
User Impact85%
